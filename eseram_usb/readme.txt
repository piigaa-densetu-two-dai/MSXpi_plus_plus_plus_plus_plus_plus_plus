ESE RAM(1024K)ファームウェアです。
FUNCTIONボタンを押すことによりRAMの内容をUSBメモリのルートディレクトリのESERAM.BINに保存することが出来ます。

ご注意
ターボRモード非対応です。

使用法

1. MSXに刺さっていない状態のMSXπ✨✨✨✨✨✨✨をブートモードでPCに接続します。

MSXπ✨✨✨✨✨✨✨をBOOTSELボタンを押しながらPCとUSB接続して下さい。
接続が成功するとドライブが認識されます。

2. MSXπ✨✨✨✨✨✨✨にeseram_usb.uf2を書き込みします。

1で認識されたドライブにeseram_usb.uf2をドラッグアンドドロップ(コピー)します。
コピーが完了するとドライブが見えなくなります。

3. 書込みが終わったらPCから外します。

4. MSXπ✨✨✨✨✨✨✨にUSBメモリを取り付けて完了

USBメモリはFAT32でフォーマットして下さい。
ディスクラベルの無いUSBメモリ(直にFAT32フォーマットされた物)は問題があるかもしれません。
ESERAM.BINファイルが無ければ自動作成されます。ESERAM.BINファイルがあればそれを使用します。

RAM保存方法
任意のタイミングでFUNCTIONボタンを押すとRAMの内容がESERAM.BINファイルに保存されます。保存中はLEDが点灯します。
保存中はRAMの内容を変更しないで下さい。

RAM復元方法
MSX起動時に自動的に復元されます。復元が完了するまでMSXは起動しません。復元中はLEDが点灯します。
MSX起動時にFUNCTIONボタンが押されていると復元処理はスキップされます。RAMは空の状態になります。
